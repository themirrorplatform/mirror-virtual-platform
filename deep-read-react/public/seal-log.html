Seal Log

Every sealed artifact in this match — the storytellers' hidden layers, every contestant submission, every revision, the crowd tally — is SHA-256 hashed and anchored via OpenTimestamps the day it is created, then logged here and mirrored as a commit in the public repository. The hash proves when. The referee's custody proves what. This log proves nothing was swapped or dropped.

LOG FORMAT (one line per seal)
[date] · [artifact id] · [sha-256] · [ots proof link] · [custodian: referee]

ENTRY 000 — this page, hashed at publication, so even the log has a log.

When the match ends, every seal opens: the plaintext publishes beside its hash so anyone can verify the pair. Until then, this page is a list of promises with timestamps — which is the only kind of promise this event makes.

Repository link and verification instructions publish here when the referee signs and the first artifact is sealed.
