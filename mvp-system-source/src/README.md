# The Mirror

**A constitutional, local-first AI reflection platform**

---

## What This Is

The Mirror is a complete, production-ready web application for sovereign personal reflection. It combines:

- **20 constitutional instruments** for multimodal capture, governance, and data sovereignty
- **Layer-based architecture** (Sovereign, Commons, Builder, Fork)
- **Zero dark patterns** - no engagement traps, no manipulation
- **Full keyboard navigation** - power user optimized
- **WCAG AAA accessibility** - screen reader ready
- **Mobile responsive** - works on any device

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

Press `Escape` at any time to close instruments.  
Press `⌘L` to explore the Layer HUD.

## Core Principles

### 1. **User Sovereignty**
Your data belongs to you. Export or delete at any time. No tracking, no analytics, no surveillance.

### 2. **No Coercion**
The system never tells you what to do. All choices have equal weight. No "recommended" actions.

### 3. **Epistemic Humility**
AI outputs are labeled as reflective, not authoritative. The system admits uncertainty.

### 4. **Transparency**
All boundaries are visible. All behavior is inspectable. Nothing happens without your knowledge.

### 5. **Silence First**
The default is a blank field. Empty states show "..." not "Get Started."
