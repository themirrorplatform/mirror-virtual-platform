## AI Response Style Comparison

### The Mirror (Mirrorback)
```
User writes: "I keep telling myself I'll change but I never do."

Mirrorback appears below (indented, bordered):
"The word 'never' appears. Is that the whole truth, or part of it?"
```

**Style:** Reflective, non-directive, pattern-showing

---

### Replika

## Data Sovereignty Comparison

## What The Mirror Does Differently

### 1. **Presence Over Onboarding**
- No setup wizard
- No "getting started" flow
- Immediate entry into core experience

### 2. **Pause-Responsive UI**
- Controls appear only when needed
- Typing = controls hidden
- Pause = controls fade in
- **No other app does this**

### 3. **Reflection Without Direction**
- AI never suggests "next steps"
- No "improvement" framing
- Shows patterns, doesn't prescribe fixes

### 4. **Hidden-Then-Revealed Navigation**
- Not "minimalist" (still full-featured)
- Not "always visible" (reduces noise)
- **Reveals on intent, not by default**

### 5. **Constitutional Governance**
- Users can propose amendments
- Fork the system to test variants
- Commons governance for shared rules
- **No other platform has this**

### 6. **Witnessing (Not Liking)**
- No metrics
- No follower counts
- No engagement optimization
- Response requires reflection on *what part* they're responding to

### 7. **Mirrorback Indentation**
- Visually distinct from user text
- Clearly labeled as reflection
- Not treated as "the answer"
- Hideable without consequence

## User Journeys: Side-by-Side

### Scenario: User wants to reflect on feeling stuck

#### The Mirror
```
1. Open app ‚Üí See writing surface
2. Type: "I feel stuck"
3. Pause ‚Üí Controls fade in
4. Click Mirror icon
5. Mirrorback appears: "What lives beneath that uncertainty?"
6. User can hide, archive, or keep typing
```

**Outcome:** Reflection deepened, no direction imposed

---

#### Replika
```
1. Open app ‚Üí Chat interface
2. Type: "I feel stuck"

5. Replika offers suggestions or support
```

**Outcome:** Emotional support, conversation-driven

---

#### ChatGPT
```
1. Open app ‚Üí Chat interface
2. Type: "I feel stuck"
3. ChatGPT: "Feeling stuck is common. Can you describe what area of life you're feeling stuck in?"
4. User elaborates
5. ChatGPT provides strategies/frameworks
```

**Outcome:** Problem-solving approach, actionable advice

---

#### Day One

1. Open app ‚Üí New entry
2. Type: "I feel stuck"
3. Continue writing freely
4. Save entry
